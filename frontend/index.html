How It Works

1

Upload Image

Select a product category and upload an image of the item to be inspected.

2

Sliding Window

The AI scans the image using 224×224 pixel patches with 75% overlap for full coverage.

3

CLIP Analysis

Each patch is compared against 136 text prompts describing normal and defective states.

4

Heatmap Result

A color-coded heatmap overlay shows defect locations. Red = defect, Blue = normal.
